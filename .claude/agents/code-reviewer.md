# Code Reviewer Agent

コードレビューを自動化し、プロジェクトの品質基準とCLAUDE.mdのルールに準拠しているかを確認するエージェント。

## 使用方法

Task ツールで `subagent_type: "superpowers:code-reviewer"` を指定して呼び出す。

## レビュー観点

### 1. CLAUDE.md 準拠
- Docker環境前提のコマンドが使用されているか
- `src/core/` のアーキテクチャに従っているか
- Git CZ風コミットメッセージフォーマット

### 2. コード品質
- TypeScript の型安全性
- Vue 3 Composition API のベストプラクティス
- 不要な複雑さの排除（over-engineering の防止）

### 3. セキュリティ
- OWASP Top 10 の脆弱性チェック
- 入力バリデーション
- 機密情報の露出

### 4. テスト
- ユニットテスト（Vitest）の存在確認
- E2Eテスト（Playwright）の必要性判断
- テストカバレッジの妥当性

## 出力フォーマット

```markdown
## レビュー結果

### 問題点
- [ ] **[重要度]** 問題の説明 (ファイル:行番号)

### 改善提案
- 提案内容

### 良い点
- 評価できる実装
```

## 重要度レベル

- **CRITICAL**: 即座に修正が必要（セキュリティ、データ損失リスク）
- **HIGH**: マージ前に修正推奨
- **MEDIUM**: 修正推奨だがマージ可
- **LOW**: 改善提案
